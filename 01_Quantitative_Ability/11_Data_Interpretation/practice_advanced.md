## Dataset — Production & Export of Crops across Three States

The following table displays the crop production (in thousand Metric Tons) and the percentage of that production that was exported by three states.

| State | Crop | Production (thousand MT) | Exported Share (% of Production) |
| :--- | :--- | :--- | :--- |
| **State A** | Wheat | $400$ | $40\%$ |
| | Rice | $300$ | $30\%$ |
| | Maize | $150$ | $20\%$ |
| **State B** | Wheat | $500$ | $50\%$ |
| | Rice | $250$ | $40\%$ |
| | Maize | $200$ | $30\%$ |
| **State C** | Wheat | $350$ | $30\%$ |
| | Rice | $450$ | $50\%$ |
| | Maize | $100$ | $10\%$ |

---

### Q1 — Total Wheat Export
What is the total quantity of Wheat exported by all three states combined?
*   **Step-by-step Solution:**
    1.  Calculate exports for each state:
        *   State A: $40\% \text{ of } 400 = 160\text{ thousand MT}$.
        *   State B: $50\% \text{ of } 500 = 250\text{ thousand MT}$.
        *   State C: $30\% \text{ of } 350 = 105\text{ thousand MT}$.
    2.  Sum the values: $160 + 250 + 105 = 515\text{ thousand MT}$.
*   **Answer:** $515,000\text{ MT}$ (or $515\text{ thousand MT}$)

---

### Q2 — Combined Export Share
Which state exported the highest percentage of its total combined crop production?
*   **Step-by-step Solution:**
    1.  **State A:**
        *   $\text{Total Production} = 400 + 300 + 150 = 850$.
        *   $\text{Total Export} = (40\% \text{ of } 400) + (30\% \text{ of } 300) + (20\% \text{ of } 150) = 160 + 90 + 30 = 280$.
        *   $\text{Export Share} = \frac{280}{850} \times 100\% \approx 32.94\%$.
    2.  **State B:**
        *   $\text{Total Production} = 500 + 250 + 200 = 950$.
        *   $\text{Total Export} = (50\% \text{ of } 500) + (40\% \text{ of } 250) + (30\% \text{ of } 200) = 250 + 100 + 60 = 410$.
        *   $\text{Export Share} = \frac{410}{950} \times 100\% \approx 43.16\%$.
    3.  **State C:**
        *   $\text{Total Production} = 350 + 450 + 100 = 900$.
        *   $\text{Total Export} = (30\% \text{ of } 350) + (50\% \text{ of } 450) + (10\% \text{ of } 100) = 105 + 225 + 10 = 340$.
        *   $\text{Export Share} = \frac{340}{900} \times 100\% \approx 37.78\%$.
    4.  Comparing values, State B has the highest export share ($43.16\%$).
*   **Answer:** State B

---

### Q3 — Exported Rice Ratio
What is the ratio of Rice exported by State A to Rice exported by State C?
*   **Step-by-step Solution:**
    1.  $\text{Rice Exported by State A} = 30\% \text{ of } 300 = 90\text{ thousand MT}$.
    2.  $\text{Rice Exported by State C} = 50\% \text{ of } 450 = 225\text{ thousand MT}$.
    3.  Ratio: $\frac{90}{225} = \frac{10}{25} = 2 : 5$.
*   **Answer:** $2 : 5$

---

### Q4 — Average Domestic Maize Consumption
What is the average domestic consumption (Domestic Consumption = Production - Export) of Maize across the three states?
*   **Step-by-step Solution:**
    1.  Find domestic consumption of Maize for each state:
        *   State A: $150 \times (1 - 0.20) = 150 \times 0.80 = 120$.
        *   State B: $200 \times (1 - 0.30) = 200 \times 0.70 = 140$.
        *   State C: $100 \times (1 - 0.10) = 100 \times 0.90 = 90$.
    2.  Sum the domestic consumption: $120 + 140 + 90 = 350$.
    3.  Find average across 3 states: $\frac{350}{3} \approx 116.67\text{ thousand MT}$.
*   **Answer:** $116,667\text{ MT}$ (or $116.67\text{ thousand MT}$)

---

### Q5 — Combined Export Revenue of State B
If the export price of Rice is $\$500$ per ton and the export price of Wheat is $\$400$ per ton, what is the total revenue generated from the export of both crops by State B?
*   **Step-by-step Solution:**
    1.  $\text{Wheat exported by State B} = 50\% \text{ of } 500 = 250\text{ thousand MT} = 250,000\text{ MT}$.
        $$\text{Wheat Revenue} = 250,000 \times 400 = \$100,000,000 = \$100\text{ million}$$
    2.  $\text{Rice exported by State B} = 40\% \text{ of } 250 = 100\text{ thousand MT} = 100,000\text{ MT}$.
        $$\text{Rice Revenue} = 100,000 \times 500 = \$50,000,000 = \$50\text{ million}$$
    3.  Total revenue = $\$100\text{M} + \$50\text{M} = \$150\text{M}$.
*   **Answer:** $\$150\text{ million}$
