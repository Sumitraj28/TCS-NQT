# Number System

The Number System is the foundation of Quantitative Ability. It covers classification of numbers, divisibility rules, remainders, factors, and common numerical patterns.

- **TCS NQT Weightage**: 🔴 High — typically 2–3 questions in the Numerical Ability section.
- **Difficulty**: Easy to Medium
- **Expected Questions**: Divisibility-based, last digit calculation, or LCM/HCF word puzzles.
