# Number System — Quick Revision

## Divisibility Quick Rules
- **3 / 9:** Digit sum.
- **4 / 8:** Last 2 digits / last 3 digits.
- **6:** Divisible by 2 and 3.
- **11:** Difference of odd/even position digit sums.

## Key Formulas

### Cyclicity exponents
- Cycle repeats every 4 powers.
- If $P \bmod 4 == 0$, use power exponent 4.

### Factors Count
- For $N = p_1^{a_1} \times p_2^{a_2} \dots$, Factors Count $= (a_1 + 1)(a_2 + 1)\dots$

### Trailing Zeros
- $\lfloor N/5 \rfloor + \lfloor N/25 \rfloor + \lfloor N/125 \rfloor \dots$

### HCF & LCM Fractions
- $\text{HCF} = \frac{\text{HCF of Numerators}}{\text{LCM of Denominators}}$
- $\text{LCM} = \frac{\text{LCM of Numerators}}{\text{HCF of Denominators}}$
