## 1. Core Principles of Object-Oriented Programming
This guide provides the foundational theory and mental models needed to master Object-Oriented Programming for the TCS NQT exam.

### Real-World Analogy:
Every system is governed by relationships. Just like variables in a software system depend on inputs to produce outputs, Object-Oriented Programming questions require you to understand how change propagates.

## 2. Core Mechanics
*   **Rule A:** Understand constraints before beginning any calculation.
*   **Rule B:** Always map out variables systematically in a table or list.
