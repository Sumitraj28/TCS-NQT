# Recursion

Recursion is a process in which a function calls itself directly or indirectly to solve a smaller subproblem. Recursion is the foundation for advanced algorithms like Dynamic Programming, backtracking, and tree traversals.

- **TCS NQT Frequency**: 🟡 Medium
- **Difficulty**: Easy to Medium
- **Expected problems per slot**: 0–1 problems, usually showing up as a recursion-based sequence or simple transition problem.
