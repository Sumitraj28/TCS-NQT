### Q1 — K-th Symbol in Grammar
We build a table of rows. Row 1 is `0`. Subsequent rows generated by replacing `0` with `01` and `1` with `10`. Find $k$-th symbol of $n$-th row.
**Full Solution:**
The first half of row $N$ is identical to row $N-1$, and the second half is the binary negation of row $N-1$.
```cpp
class Solution {
public:
    int kthGrammar(int n, int k) {
        if (n == 1) return 0;
        int mid = 1 << (n - 2);
        if (k <= mid) return kthGrammar(n - 1, k);
        return !kthGrammar(n - 1, k - mid);
    }
};
```
**Complexity:** Time $O(N)$ | Space $O(N)$

### Q2 — Generate Parentheses
Given $N$ pairs, generate all combinations of well-formed parentheses.
**Full Solution (Backtracking DFS):**
```cpp
#include <vector>
#include <string>

class Solution {
private:
    void backtrack(std::vector<std::string>& res, std::string current, int open, int close, int max) {
        if (current.length() == max * 2) {
            res.push_back(current);
            return;
        }
        if (open < max) {
            backtrack(res, current + "(", open + 1, close, max);
        }
        if (close < open) {
            backtrack(res, current + ")", open, close + 1, max);
        }
    }
public:
    std::vector<std::string> generateParenthesis(int n) {
        std::vector<std::string> res;
        backtrack(res, "", 0, 0, n);
        return res;
    }
};
```
**Complexity:** Time $O(\frac{4^N}{\sqrt{N}})$ | Space $O(N)$

### Q4 — Josephus Problem
$N$ people standing in a circle, every $K$-th person is executed. Find survivor index.
**Full Solution:**
```cpp
int josephus(int n, int k) {
    if (n == 1) return 0;
    return (josephus(n - 1, k) + k) % n;
}
```
**Complexity:** Time $O(N)$ | Space $O(N)$

### Q5 — Subsets Generator
Given integer array `nums`, return all possible subsets (power set).
**Full Solution (Backtracking):**
```cpp
#include <vector>

void backtrack(std::vector<std::vector<int>>& res, std::vector<int>& temp, const std::vector<int>& nums, int start) {
    res.push_back(temp);
    for (size_t i = start; i < nums.size(); ++i) {
        temp.push_back(nums[i]);
        backtrack(res, temp, nums, i + 1);
        temp.pop_back();
    }
}
std::vector<std::vector<int>> subsets(const std::vector<int>& nums) {
    std::vector<std::vector<int>> res;
    std::vector<int> temp;
    backtrack(res, temp, nums, 0);
    return res;
}
```

### Q10 — Word Search
Find if a word exists in a 2D board (DFS grid traversal).
**Full Solution:** Cell-by-cell matching recursive grid search with visited markers.
```cpp
#include <vector>
#include <string>

bool dfs(std::vector<std::vector<char>>& board, const std::string& word, int i, int j, int k) {
    if (k == word.length()) return true;
    if (i < 0 || i >= board.size() || j < 0 || j >= board[0].size() || board[i][j] != word[k]) return false;
    
    char temp = board[i][j];
    board[i][j] = '\0'; // mark visited
    bool res = dfs(board, word, i + 1, j, k + 1) ||
               dfs(board, word, i - 1, j, k + 1) ||
               dfs(board, word, i, j + 1, k + 1) ||
               dfs(board, word, i, j - 1, k + 1);
    board[i][j] = temp; // restore
    return res;
}
```
**Complexity:** Time $O(M \times N \times 4^L)$ where $L$ is word length | Space O(L)
Ref: standard DFS backtracker.
