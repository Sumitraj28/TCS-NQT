# Easy Dynamic Programming (DP)

Dynamic Programming is an algorithmic technique for solving optimization and counting problems by breaking them down into simpler subproblems and utilizing the fact that the optimal solution to the overall problem depends upon the optimal solutions to its subproblems.

- **TCS NQT Frequency**: 🔴 High — very common as Problem 2 (Medium) or Prime questions
- **Difficulty**: Medium
- **Expected problems per slot**: 1 DP problem (Climbing Stairs variants, Fibonacci variants, robber problems).
