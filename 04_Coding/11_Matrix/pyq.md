# Matrix — Previous Year Style Questions (TCS NQT)

---

### PYQ1 — Spiral Traversal
**Difficulty:** Medium | **Time:** 90s | **TCS Frequency:** High
Find spiral path traversal of matrix.
**C++ Solution:** Boundaries reduction `top`, `bottom`, `left`, `right`.

---

### PYQ2 — Rotate Matrix 90 Degrees Clockwise
**Difficulty:** Medium | **Time:** 60s | **TCS Frequency:** High
Rotate square matrix in-place.
**C++ Solution:** Swap `matrix[i][j]` with `matrix[j][i]` (transpose), reverse each row.

---

### PYQ3 — Search Row/Col Sorted Matrix
**Difficulty:** Medium | **Time:** 60s | **TCS Frequency:** High
Check if target value exists in sorted grid.
**C++ Solution:** Top-right pointer movement technique.

---

### PYQ4 — Set Matrix Zeroes
**Difficulty:** Medium | **Time:** 90s | **TCS Frequency:** High
**C++ Solution:** Use row 0 and col 0 as marker arrays.

---

### PYQ5 — Diagonal Traverse
**Difficulty:** Medium | **Time:** 90s | **TCS Frequency:** High
Print elements diagonally.
**C++ Solution:** Loop index sum and swap directions.
