# Strings

Strings are sequence of characters and represent one of the most frequently asked topics in TCS NQT.
This section covers essential string manipulation patterns including Two Pointer reversals, sliding windows, Anagram checks, and custom parser logic (atoi).

- **TCS NQT Frequency**: 🔴 High
- **Difficulty**: Easy to Medium
- **Expected problems per slot**: 1 string problem is very common
