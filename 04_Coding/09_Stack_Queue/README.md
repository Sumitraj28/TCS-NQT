# Stacks & Queues

Stacks and Queues are linear data structures with restricted insertion and deletion operations. A stack follows LIFO (Last In First Out), whereas a queue follows FIFO (First In First Out).

- **TCS NQT Frequency**: 🔴 High
- **Difficulty**: Easy to Medium
- **Expected problems per slot**: 1 problem involving parsing, matching (brackets), or sliding window queries.
