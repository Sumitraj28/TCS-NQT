# HashMap

HashMap (implemented via `std::unordered_map` and `std::unordered_set` in C++) is the most critical data structure for achieving O(1) average lookup times.
This section covers frequency counting, duplicate detection, array intersection, grouping by key (anagram grouping), and prefix sum mapping.

- **Difficulty**: Easy to Medium
