# Searching

Searching covers finding elements in arrays — from simple linear scan to the elegant O(log n) binary search.
TCS NQT tests binary search not just as "find x in sorted array" but also through its variants:
Lower Bound, Upper Bound, First/Last Occurrence, and search on answer-space (binary search on the answer).

- **TCS NQT Frequency**: 🔴 High
- **Difficulty**: Easy (Linear/Basic Binary) to Medium (Variants / Answer-Space Binary Search)
- **Expected**: At least one searching problem per coding slot; variants appear in Advanced Aptitude too
