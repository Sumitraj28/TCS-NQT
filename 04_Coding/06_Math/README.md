# Math

Math covers fundamental arithmetic, number theory, prime numbers, factors, GCD/LCM, and digit manipulation. These forms are critical in the TCS NQT coding section, where many numerical puzzles are asked.

- **TCS NQT Frequency**: 🔴 High — very common as problem 1 (Easy) or as a helper in problem 2
- **Difficulty**: Easy to Medium
- **Expected problems per slot**: 1 numerical or digit-based coding problem
