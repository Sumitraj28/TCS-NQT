# Linked List — Previous Year Style Questions (TCS NQT)

---

### PYQ1 — Reverse Singly Linked List
**Difficulty:** Easy | **Time:** 60s | **TCS Frequency:** High
Reverse singly linked list.
**C++ Solution:** Change pointers iteratively using three variables.

---

### PYQ2 — Find Middle Node
**Difficulty:** Easy | **Time:** 45s | **TCS Frequency:** High
Find middle of linked list in a single pass.
**C++ Solution:** Slow and fast pointer strategy.

---

### PYQ3 — Linked List Cycle Check
**Difficulty:** Easy | **Time:** 45s | **TCS Frequency:** High
Check if list has cycle.
**C++ Solution:** Floyd's Tortoise and Hare pointers.

---

### PYQ4 — Merge Two Sorted Lists
**Difficulty:** Medium | **Time:** 75s | **TCS Frequency:** High
Merge lists.
**C++ Solution:** Compare node values iteratively using dummy head.

---

### PYQ5 — Remove N-th Node From End
**Difficulty:** Medium | **Time:** 90s | **TCS Frequency:** High
Delete the $N$-th node from the end of the list.
**C++ Solution:** Maintain a gap of $N$ nodes between two pointers, then delete the node.
