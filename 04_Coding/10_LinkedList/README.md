# Linked Lists

A Linked List is a linear data structure where elements are stored in nodes, and each node points to the next node in memory using pointers. Unlike arrays, elements are not stored contiguously.

- **TCS NQT Frequency**: 🔴 High
- **Difficulty**: Easy to Medium
- **Expected problems per slot**: 1 problem involving reversal, cycle checks, or node removals/merging.
