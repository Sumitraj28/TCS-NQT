# Sorting

Sorting is the process of arranging elements in a defined order (ascending/descending). Understanding the
mechanics of Bubble, Selection, Insertion, Merge, and Quick Sort — plus knowing when to use Java/Python
built-ins — is essential for TCS NQT coding problems. Nearly every medium-hard problem uses sorting
as a preprocessing step.

- **TCS NQT Frequency**: 🔴 High (as a sub-step in almost all problems)
- **Difficulty**: Easy (concepts) to Medium (applying in problems)
- **Direct coding problems**: Sort Colors, Kth Largest, Merge Intervals — all appear in TCS
