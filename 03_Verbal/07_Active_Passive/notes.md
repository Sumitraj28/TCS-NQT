# Active & Passive Voice — Complete Notes

---

## 1. Core Conversion Framework

In the active voice, the subject performs the action. In the passive voice, the subject receives the action.

$$\text{Active: } \text{Subject (S)} + \text{Verb (V)} + \text{Object (O)}$$

$$\text{Passive: } \text{Object (O)} + \text{Auxiliary Verb (Be)} + \text{Past Participle (V3)} + \text{by} + \text{Subject (S)}$$

### Key Shifts
1.  **Subject-Object Swap:** The active object becomes the passive subject.
2.  **Verb Form:** The main verb is **always** converted into its **Past Participle (V3)** form in the passive voice.
3.  **Pronoun Shifts:**
    *   *I* $\rightarrow$ *me*
    *   *We* $\rightarrow$ *us*
    *   *He* $\rightarrow$ *him*
    *   *She* $\rightarrow$ *her*
    *   *They* $\rightarrow$ *them*

## 2. Tense-Wise Conversion Table

Below is the reference table showing active-to-passive conversions across all standard tenses.

| Tense | Active Structure | Passive Structure | Active Example | Passive Example |
| :--- | :--- | :--- | :--- | :--- |
| **Simple Present** | S + V1 (s/es) + O | O + is/am/are + V3 + by S | He writes a letter. | A letter **is written** by him. |
| **Present Continuous** | S + is/am/are + V1-ing + O | O + is/am/are + **being** + V3 + by S | He is writing a letter. | A letter **is being written** by him. |
| **Present Perfect** | S + has/have + V3 + O | O + has/have + **been** + V3 + by S | He has written a letter. | A letter **has been written** by him. |
| **Simple Past** | S + V2 + O | O + was/were + V3 + by S | He wrote a letter. | A letter **was written** by him. |
| **Past Continuous** | S + was/were + V1-ing + O | O + was/were + **being** + V3 + by S | He was writing a letter. | A letter **was being written** by him. |
| **Past Perfect** | S + had + V3 + O | O + had + **been** + V3 + by S | He had written a letter. | A letter **had been written** by him. |
| **Simple Future** | S + will/shall + V1 + O | O + will/shall + **be** + V3 + by S | He will write a letter. | A letter **will be written** by him. |
| **Future Perfect** | S + will have + V3 + O | O + will have + **been** + V3 + by S | He will have written a letter. | A letter **will have been written** by him. |

*Note: Present Perfect Continuous, Past Perfect Continuous, Future Continuous, and Future Perfect Continuous tenses do not have standard passive conversions.*

## 3. Special Cases (Imperative & Interrogative)

### A. Imperative Sentences (Orders, Requests)
*   **Order/Command structure:**
    *   *Active:* Write the letter.
    *   *Passive (Formula: Let + Object + be + V3):* **Let** the letter **be written**.
*   **Request structure:**
    *   *Active:* Please sit down.
    *   *Passive:* You are requested to sit down.

### B. Interrogative Sentences (Questions)
*   If the question starts with "Who", it converts to "By whom" in the passive voice:
    *   *Active:* Who wrote this letter?
    *   *Passive:* **By whom** was this letter written?
