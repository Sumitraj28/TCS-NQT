# Sliding Window - Formulas & Derivations

This file contains key mathematical models and derivations for Sliding Window.

### 📐 Formula 1: Basic Relationship
$$\text{Value} = \text{Constant} \times \text{Variable}$$
*   **Derivation:** The output scales directly with the variable parameter when the system constants are locked.
*   **Worked Example:** Let $\text{Constant} = 4$, $\text{Variable} = 25$.
    $$\text{Value} = 4 \times 25 = 100$$
