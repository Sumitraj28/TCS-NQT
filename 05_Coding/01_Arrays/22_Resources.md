## ✏️ Curated Practice Problems List

Complete these ten problems to cover the most common array patterns tested in the TCS NQT:

### Two-Pointer & Partition Patterns
- [x] **[Two Sum (LeetCode 1)](https://leetcode.com/problems/two-sum/)**: Practice hashing or two-pointer target matching.
- [x] **[Sort Colors / Sort 012 (LeetCode 75)](https://leetcode.com/problems/sort-colors/)**: Dutch National Flag partition algorithm.
- [x] **[Container With Most Water (LeetCode 11)](https://leetcode.com/problems/container-with-most-water/)**: Two-pointer boundary contraction.

### Sliding Window & Subarray Patterns
- [x] **[Maximum Subarray Sum (LeetCode 53)](https://leetcode.com/problems/maximum-subarray/)**: Standard Kadane's algorithm.
- [x] **[Subarray Sum Equals K (LeetCode 560)](https://leetcode.com/problems/subarray-sum-equals-k/)**: Prefix sum + hashing map integration.
- [x] **[Longest Substring Without Repeating Characters (LeetCode 3)](https://leetcode.com/problems/longest-substring-without-repeating-characters/)**: Sliding window with character frequency tracking.

### Hard & Advanced Patterns
- [x] **[Trapping Rain Water (LeetCode 42)](https://leetcode.com/problems/trapping-rain-water/)**: Two-pointer peak tracking.
- [x] **[Merge Intervals (LeetCode 56)](https://leetcode.com/problems/merge-intervals/)**: Array sorting combined with interval overlaps.
- [x] **[First Missing Positive (LeetCode 41)](https://leetcode.com/problems/first-missing-positive/)**: Index-as-hash cycle sort pattern.
