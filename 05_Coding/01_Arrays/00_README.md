# 05 Coding - Arrays Directory Overview

Welcome to the **Arrays** module of the TCS NQT coding preparation space. This directory contains theory, visual guides, problem templates, and compile-ready C++14 solutions to solve Array problems in the actual exam.

---

## 💻 DS/Algo Pattern: Arrays
An array is a contiguous memory structure storing elements of the same data type. Most TCS NQT coding exams contain at least one question directly requiring array traversal, pointer manipulation, sliding window, or prefix sum analysis.

---

## 🎯 Why TCS NQT Tests It
TCS tests arrays to verify:
1.  **Memory Management**: Index bounding and avoiding segmentation faults.
2.  **Algorithm Optimization**: Transitioning from brute-force $O(N^2)$ loops to $O(N)$ linear scans using Two Pointers or Hashing.
3.  **Data Manipulation**: Shifting, partitioning, or rotating structures under pressure.
